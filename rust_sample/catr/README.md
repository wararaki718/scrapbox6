# catr

## build

```shell
cargo build
```

## run

```shell
cargo run -- --help
```

```shell
cargo run -- tests/input/fox.txt
```

```shell
cargo run -- - < tests/input/fox.txt
```

```shell
cargo run -- -n tests/input/spiders.txt
```

```shell
cargo run -- -b tests/input/the-bustle.txt
```

## test

```shell
cargo test
```
